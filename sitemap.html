網站導覽說明

本網站依無障礙網頁設計原則而建置，網站的主要內容分為三個大區塊：

1.上方功能區塊

2.左方選單連結區

3.中央內容區

本網站的便捷鍵﹝Accesskey，也稱為快速鍵﹞設定如下：

Alt + U：上方功能區，包括回首頁、網站導覽。

Alt + N：主選單連結區。

Alt + L：左方選單連結區。

Alt + C：中央內容區，為本頁主要內容區。

Alt + Z：下方功能區，包含服務專線、專線服務時間及建議瀏覽器使用版本。

Ctrl + W：關閉視窗。

以上適用瀏覽器:IE 9/10，Chrome、Safari、Firefox(請搭配Shift+Alt使用)。

當本網站項目頁籤無法以滑鼠點選時，您可利用以下鍵盤操作方式瀏覽資料：

1. 您可以使用 Tab 鍵跳至下一個點選項目，並利用 Enter 鍵執行點選的動作。

2. 您可以使用 Alt + F4 關閉視窗。

3. 您可以使用 Home 及 End 直接跳至標籤第一項或者最後一項。

4. 您可以使用 Tab + Shift 往回跳至上一筆資料。
